# Design spec

Brand name: LeadKast. Domain, Twitter, and Instagram handles appear available (preliminary web search confirmed; formal trademark search required before filing).

- Visual identity: Dark primary palette (#1A1A2E) with accent red (#E94560) and amber (#F5A623). Fonts are Inter (body) and Space Grotesk (display). The tone is systematic, results-driven, discreet, operator-focused, and data-backed.
- Key screens: (1) Dashboard / Command Center showing live KPIs (spend, CPL, leads, contracts, cost-per-contract) with quick-launch buttons. (2) Lead Pipeline Board — Kanban-style drag-and-drop with stages: New Lead → DM Sent → Typeform Completed → Sales Call Booked → Contract Sent → Signed. (3) Ad Creative Library — grid of downloadable tested creatives tagged by format, region, and performance tier. (4) Campaign Setup Wizard — 6-step guided flow for operators with no Meta Ads experience. (5) Reporting & ROI Dashboard — spend-vs-leads charts, CPL trends, and exportable PDF reports.
- Every screen has designed empty states (encouraging first action), loading states (skeleton/shimmer patterns), and error states (amber banners with clear recovery actions).
- Example social post tone: '81 leads. 3 signed contracts. $374 spent. 2 weeks. This is what a tested model recruitment system looks like.'
